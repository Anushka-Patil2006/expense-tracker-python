# Student Expense Tracker (Python)

## Description
This is a beginner-level Python project created to practice basic programming
concepts such as loops, functions, and file handling.

## Features
- Add daily expenses
- Store expenses in a CSV file
- View all expenses
- Calculate total expense

## Tech Stack
- Python
- CSV file

## How to Run
1. Open the project folder
2. Run expense_tracker.py using Python IDLE or Command Prompt
3. Follow the menu options

## What I Learned
- Python basics
- File handling using CSV
- Writing clean and readable code
